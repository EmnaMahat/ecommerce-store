# E‑Commerce Store API

A RESTful API built with **Node.js**, **Express**, and **MongoDB Atlas** to manage an online product catalog.

## 🚀 Features

- Create a new product
- Retrieve all products
- Retrieve a product by ID
- Update a product
- Delete a product
- Automated testing with Mocha/Chai
- Deployment on Render

---

## 📦 Installation

1. Clone the repository:
   git clone https://github.com/EmnaMahat/ecommerce-store.git

2. Install dependencies:
   npm install

3. Create a `.env` file:
   MONGO_URI=your_mongodb_connection_string

4. Start the server:
   npm start

---

## 🌐 API Endpoints

### GET /api/products  
Returns all products.

### POST /api/products  
Creates a new product.

### GET /api/products/:id  
Returns a product by its ID.

### PUT /api/products/:id  
Updates a product.

### DELETE /api/products/:id  
Deletes a product.

---
